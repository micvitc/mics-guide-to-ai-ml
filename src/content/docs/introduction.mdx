Welcome to MIC's Guide to AI/ML. This resource is meant to be a beginner friendly and simple guide to start you Machine Learning journey. It consists of several tutorials ranging from introductions to different frameworks to end to end project implementations. We'll explore different fields of machine learning ranging from Computer Vision, Natural Language Processing to Reinforcement Learning and much more. With an emphasis on real world applications and proven Software Engineering principles, along with the core ML topics we'll explore how to develop, deploy and maintain ML based applications.

All you need is basic Python skills and you are ready go. All the related concepts will be covered along the way in a simple and accessible format. Hope you have as much fun going through this guide as we did creating it. 

Good Luck!
